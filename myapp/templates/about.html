Task Management System

This repository contains a simple task management system implemented using Django. The task management system provides a user-friendly graphical user interface (GUI) for managing tasks, including adding new tasks, viewing existing tasks, and Updating tasks also deleting the tasks.

FUTURS

Add New Task: Easily add new tasks to the system using a "add task". View Tasks: View the list of tasks in a separate dialog window. Update task : We can also update the excisting tasks. History: We can store delete tasks in history. Delete all Restore all:We cav restore or delete all at one click.

Usage

Upon launching the application, a main window will appear, providing access to various

Click the "Add New Task" button to add a new task Click the "Update" button to update the task Click the "Delete" button to delete task Click 'History' will see delete tasks Click "Restore" to Restore the perticular task Click the "Restore All" button to restore all tasks at time Click the "Delete All" button to delete all task at time
